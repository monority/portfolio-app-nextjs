# VISUAL CALIBRATION — Ronan Chenu Portfolio

## VERSION A — MINIMAL (One Screen)

### Ratio Content/Whitespace
- 50% content, 50% whitespace (equilibré)
- Tout sur un écran = pas de choice

### Typography
- Font: 1 seule (system ou Google Fonts simple)
- Sizes: 3-4 max (H1: 32-48px, body: 16px, small: 14px)
- Line-height: 1.4 (plus compact)
- Letter-spacing: normal

### Colors
- Strict monochrome
- Background: #f7f6f3
- Text: #1a1a18
- Muted: #6b6b6b
- NO accent color

### UI
- Pas de bordures
- Pas d'ombres
- Pas de backgrounds colored
- List-style projects (texte seulement)

### Spacing
- Uniforme: même gap partout (24px)
- Grid strict

---

## VERSION B — DESIGN/ARTIST (Creative)

### Ratio Content/Whitespace
- 70% whitespace minimum
- 30% content maximum
- Exception: hero avec 80% whitespace

### Typography
- Dominante: 1 font family expressive
- Hierarchy: 4 niveaux max (h1, h2, h3, p)
- Line-height: 1.2 (headings), 1.6 (body)
- Letter-spacing: tighter sur headings (-0.02em à -0.03em)
- Sizes: 12, 14, 16, 20, 24, 32, 40, 48, 64, 96px

### Colors
- Background: #f7f6f3 (warm white)
- Text: #1a1a18 (near black)
- Muted: #6b6b6b (gray)
- Accent: 1 couleur max (choisie selon projet)
- NEVER: gradients, saturated colors

### Minimal UI
- Pas de bordures inutiles
- Pas d'ombres except subtle drop shadow (0 2px 8px rgba(0,0,0,0.05))
- Pas de gradients
- Clean, editorial, premium

### Feeling
- Calm: pas d'éléments jumpy
- Editorial: comme un magazine de design
- Premium: chaque pixel compte
- High-end: pas de patterns "SaaS"
- Emotionnel: raconte une histoire

### Spacing System
- Base unit: 8px
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px
- 3xl: 64px
- 4xl: 80px
- 5xl: 120px
- TOUJOURS utiliser des multiples de 8

### Rhythm
- Varier les espacements (petit/grand/petit/grand)
- Créer du flux visuel
- Pas de monotonie
- Penser "musique" — vagues et silences
