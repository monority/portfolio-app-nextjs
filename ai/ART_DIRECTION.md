# ART DIRECTION — Ronan Chenu Portfolio

## DEUX VISIONS

---

## VERSION A — MINIMAL (One Screen)

### Intent
- Ultra minimal
- Fonctionnel
- Professionnel
- CV moderne

### Feeling
- Clean
- Épuré
- Sans distraction
- Efficiency

### Visual Language
- Une seule font
- Typographie fonctionnelle (pas expressive)
- Maximum 4 éléments sur la page
- Grille stricte, alignée

### Règles
- Tout visible sur 100vh
- Pas de scroll
- Pas d'images grandes (juste miniatures ou texte)
- Pas d'animations
- Espace blanc uniforme

### Sources d'inspiration
- CV en ligne type
- Portfolios minimalistes
- Documentation technique élégante

---

## VERSION B — DESIGN/ARTIST (Creative)

### Intent
- Editorial
- Précoce
- Minimal
- Premium
- Artistique

### Feeling
- Calm (pas de bruit)
- Controlled (tout intentional)
- High-end (awwwards level)
- Modern (actuel, pas daté)
- Emotionnel (raconte une histoire)

### Visual Language
- Typographie dominante et expressive
- Grand whitespace (70-80%)
- Hiérarchie forte
- Une couleur d'accent
- Layout créatif (asymétrie possible)

### Design Philosophy
- "If in doubt, leave it out"
- Remove everything unnecessary
- Focus on rhythm and balance
- Every pixel has meaning
- Storytelling through design

### Typographie
- Font principale : Inter (light/regular/bold)
- Limit weights: light + regular + medium + bold
- Size scale: 14, 16, 24, 32, 48, 64, 96px
- Letter-spacing: tighter sur headlines (-0.02em)
- Line-height: 1.4 (headings), 1.6 (body)

### Couleurs
- Background: #f7f6f3 (warm white)
- Text: #1a1a18 (near black)
- Muted: #6b6b6b (gray)
- Accent: #0066cc (bleu pour liens), #ffd700 (jaune pour hover)
- NEVER: gradients, too many colors, bright饱和 colors

### Animations & Interactions
- ScrollTrigger GSAP pour reveal (opacity 0→1, y:50→0, ease "power2.out")
- Hover rocket launch Framer Motion
- Modals détails projets (overlay sombre, close button)
- Cursor custom

### Forbidden (les deux versions)
- Gradients everywhere
- Overly rounded corners (max 8px)
- Complex animations
- Unnecessary modals
- Heavy frameworks
- SaaS look (templates type SaaS)
