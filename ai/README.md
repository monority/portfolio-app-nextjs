# README — Portfolio Ronan Chenu

## Objectif
Produire deux versions du portfolio :
1. **MINIMAL** — One screen, ultra épuré, tout sur une page, comme un CV moderne
2. **DESIGN/ARTIST** — Style créatif senior, typographie forte, émotions, référence Salman GSamar

## Workflow
1. Charger TOUS les fichiers .md + DESIGN_TOKENS.json dans l'IA
2. Charger aussi OneScreenMinimal.tsx comme référence de la version minimal
3. Charger une URL de référence pour le style design (https://salmangsamar.com/)
4. Générer le composant en précisant EXACTEMENT quelle version

## Version MINIMAL (One Screen)
- TOUT sur un seul écran (100vh)
- Header + Hero + Projects + Contact = sans scroll
- Typographie: 1 font, sizes fixes
- Couleurs: monochrome (noir/blanc/gris)
- Spacing: uniforme, structuré
- Aucune animation ou subtile uniquement

## Version DESIGN/ARTIST (Creative)
- Scroll allowed, immersive
- Typographie: expressive, plusieurs weights
- Couleurs: palette affirmée, accent color
- Spacing: rhythmique, varié
- Animations: présentes mais subtiles
- Feelings: émotion, storytelling, art direction
